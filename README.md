# 🚀 API Service - FastAPI RESTful 接口服务

轻量级 RESTful API 服务，支持数据查询、分页、导出。

## ✨ 功能特性

- RESTful API 设计
- 数据查询 + 分页
- JSON / CSV 导出
- 数据清洗、格式化、统计
- Swagger 自动文档（/docs）
- Docker 本地部署

## 🚀 快速开始

```bash
pip install -r requirements.txt
uvicorn main:app --reload
# 访问 http://localhost:8000/docs 查看 API 文档
```

## 📦 技术栈

- Python 3.10+
- FastAPI / uvicorn
- pandas
